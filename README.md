## 🎯 Features

- **Brutalist Design** — Heavy borders, raw typography, concrete textures, and high-contrast colors
- **Fully Responsive** — Adapts seamlessly from desktop to mobile
- **Custom Cursor** — Interactive square cursor that responds to hover states
- **Scroll Animations** — Elements reveal themselves as you scroll
- **Sticky Navigation** — Color-inverts on scroll for visual impact
- **Booking Modal** — Functional appointment form with success state
- **Pricing Table** — Clean, brutalist-styled service menu
- **Zero Dependencies** — Pure HTML, CSS, and vanilla JS
